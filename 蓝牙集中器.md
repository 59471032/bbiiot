# NB 远传功能
集中器具有NBiot（电信）接入功能
电信卡要设置成常在线模式  psm0模式
# 档案设置
集中器具有蓝牙接口，可以提供外围设置功能
具备增删档案功能

# 485抄表功能
- 需要支持1-2路com 口抄表。
- 最大支持32个表

# 任务管理
- 任务分为3类，1小时任务（4条15分钟曲线）；2小时任务（4条30分钟曲线）；4小时任务（4条1小时数据）
